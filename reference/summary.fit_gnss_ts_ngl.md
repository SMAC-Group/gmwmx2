# Extract estimated parameters from a `fit_gnss_ts_ngl`

Extract estimated parameters from a `fit_gnss_ts_ngl`

## Usage

``` r
# S3 method for class 'fit_gnss_ts_ngl'
summary(object, scale_parameters = FALSE, ...)
```

## Arguments

- object:

  A `fit_gnss_ts_ngl` object.

- scale_parameters:

  A `boolean` indicating whether or not to scale estimated parameters so
  that the returned estimated trend is provided in m/year instead of
  m/day. Default is FALSE.

- ...:

  Additional parameters.
